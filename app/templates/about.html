a criação do plenário

O plenário é uma ferramenta open source para o controle social. Esta versão inicial apresenta os dados de votações do Senado.

O site de monitoramento online procura unir interesse público, transparência e visualização de dados.

O projeto foi desenvolvido como trabalho final para o curso bootcamp {reprograma}, com apoio do Senai de Santa Cecília, unidade localizada em São Paulo.
